This version of Pigale uses Qt version 5.14.

The directorie pigaleXXX contains: 
	pigale.app, client.app, pigale-cgi ClientData.txt		
and the directories	
images:											
    some Archimedean graphs.
Doc:
  	HTML documentation
macro:
	a macro example
tgf:
	graph examples (.txt, .tgf, .graphml)
	
Pigale settings are saved in the file:
/Users/USERNAME/Library/Preferences/com.ehess-cnrs.Pigale.plist

Pigale log file:
/Users/USERNAME/pigalelog.txt
