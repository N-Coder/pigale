This version of Pigale uses Qt version 5.13.2
This Qt version is compatible Windows 10. 

The directorie pigaleXXX contains: 
	ClientData.txt
and the directories
bin:
	pigale.exe, client.exe and all the needed dll.
Doc:
  	HTML documentation
images:
    an example of a 3d embedding  	
macro:
	an example of a macro
tgf:
	graph examples (.txt, .tgf, .graphml)
	
For the system lo locate the dll's, you may have to add xxx\pigale\bin to the system path.

Pigale log file:
pigalelog.txt in the same directory as pigale.exe
